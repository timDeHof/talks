# Engineering Software Integrity - Slidev Presentation

This is a Slidev presentation converted from the original PDF slidedeck.

## About

This presentation showcases Timothy DeHof's unique approach to software engineering, applying mechanical engineering principles to build robust, scalable, and maintainable software systems.

## Getting Started

To start the slide show:

- `pnpm install` (already done)
- `pnpm dev`
- visit <http://localhost:3030>

Edit the [slides.md](./slides.md) to see the changes.

## Features

- 13 slides covering engineering principles applied to software
- Interactive navigation
- Responsive design
- Professional Seriph theme
- Background images from original PDF

## Export Options

To export slides to PDF:

```bash
pnpm run export
```

To export slides to PNG images:

```bash
pnpm run export --format png
```
